> **AI-powered WhatsApp Bot | OpenAI GPT, Weather, Jokes, Quotes & More**

Intelligent WhatsApp bot with AI conversations, weather updates, jokes, motivational quotes, and useful utilities. Perfect for personal or business automation.

## ✨ Features

- 🤖 **AI Chat** - Powered by OpenAI GPT-3.5
- 🌤️ **Weather** - Real-time weather updates
- 😂 **Jokes & Quotes** - Daily entertainment
- 🔍 **Web Search** - Quick information lookup
- 📰 **News** - Latest headlines
- 🎲 **Fun Commands** - Dice, memes, facts
- 👥 **Group Support** - Works in groups
- 🔒 **Admin Controls** - Permission system

## 🚀 Installation

```bash
git clone https://github.com/DAXXTEAM/whatsapp-ai-bot.git
cd whatsapp-ai-bot
npm install
```

## 🔧 Configuration

```javascript
// config.js
module.exports = {
  OPENAI_API_KEY: "your-api-key",
  BOT_NAME: "JEEVES",
  PREFIX: "!",
  OWNER: "919876543210"
}
```

## 📖 Commands

```
!ai <message>     - Chat with AI
!weather <city>   - Get weather
!joke             - Random joke
!quote            - Motivational quote
!news             - Latest news
!meme             - Random meme
!help             - Show all commands
```
